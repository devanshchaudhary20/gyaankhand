# Audio tracks

Drop royalty-free `.mp3` or `.m4a` files here. One is picked at random each day
and mixed into the Reel video before upload.

## What works well

- Tanpura or shruti box drone (loopable, neutral)
- Sitar / bansuri ambient loops
- Soft temple bell ambience

## Royalty-free sources

- [Pixabay](https://pixabay.com/music/) — search "tanpura", "sitar", "indian meditation"
- [Free Music Archive](https://freemusicarchive.org/) — filter by CC0 or CC-BY
- [ccMixter](https://ccmixter.org/)

## Notes

- Any length works; ffmpeg loops the audio to match the video duration.
- Aim for tracks without sudden starts/stops for best looping.
- Keep file names simple (no spaces).
